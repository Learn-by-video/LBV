<h2>Key Examples with Comments:</h2>

<!-- BASIC BULLET POINTS -->
<p><strong>Bullet Points (Unordered List):</strong> Use <code>&lt;ul&gt;</code> and <code>&lt;li&gt;</code>.</p>
<ul>
  <li>Solar Energy</li>
  <li>Wind Energy</li>
  <li>Hydropower</li>
</ul>

<!-- DIFFERENT BULLET STYLES -->
<p><strong>Different Bullet Styles:</strong> You can change bullet types using CSS.</p>

</ul>

<!-- ORDERED LIST TYPES -->
<p><strong>Numbered Lists (Ordered List):</strong> Use <code>&lt;ol&gt;</code>.</p>
<ol>
  <li>Step One</li>
  <li>Step Two</li>
</ol>

<p><strong>Different Number Styles:</strong></p>
<ol type="1">
  <li>Numbers</li>
</ol>

<ol type="A">
  <li>Capital Letters</li>
</ol>

<ol type="a">
  <li>Lowercase Letters</li>
</ol>

<ol type="I">
  <li>Roman Numerals</li>
</ol>

<ol type="i">
  <li>Lowercase Roman Numerals</li>
</ol>

<p><strong>Start at a Different Number:</strong></p>
<ol start="5">

  <li>Then 6</li>
</ol>

<!-- REVERSED LIST -->
<p><strong>Reversed List:</strong></p>
<ol reversed>
  <li>Last Step</li>
  <li>First Step</li>
</ol>

<!-- NESTED LISTS -->
<p><strong>Nested Lists (Lists inside lists):</strong></p>
<ul>
  <li>Renewable Energy
    <ul>
      <li>Solar</li>
      <li>Wind</li>
      <li>Hydro</li>
    </ul>
  </li>
</ul>

<!-- MIXED LIST -->
<p><strong>Mixed Lists:</strong></p>
<ol>
  <li>Main Step
    <ul>
      <li>Sub-point</li>
      <li>Sub-point</li>
    </ul>
  </li>
</ol>

<!-- DESCRIPTION LIST -->
<p><strong>Description List:</strong> Use <code>&lt;dl&gt;</code>, <code>&lt;dt&gt;</code>, and <code>&lt;dd&gt;</code>.</p>
<dl>
  <dt>Solar Energy</dt>
  <dd>Energy from the sun</dd>

  <dt>Wind Energy</dt>
  <dd>Energy from moving air</dd>
</dl>
